Aplicatia se numeste Book Library si a fost creata cu ajutorul SqLite, iar pentru a se putea vedea continutul basei de date
s-a utilizat programul DB Browser for SQLite. 
Interfata aplicatiei este foarte intuitiva, astfel ca in momentul in care este deschisa apar 
datele introduse anterior sub forma unor arrayuri. 
Prima pagina a aplicatiei corespunde clasei MainActivity.java, iar urmatoarea pagina(pagina
in care utilizatorului ii este permisa introducerea datelor) corespunde clasei AddActivity. 
Prima pagina contine un buton(add_button) care face legatura cu urmatoarea pagina in care se
introduc datele.
